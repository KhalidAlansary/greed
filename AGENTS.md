# Project details

This is a school project. The project is a simple charity management app built
with C#, WinForms, ADO.net, and SQL Server.

# Preferred tools

When asked to create a document, use typst and place the document under `docs/`.

# Assignment details

## Part 1: Database Architecture

### Design

the Schema: Create a relational database with multiple tables relevant to
your project theme.

### Initial

Data Seeding: Populate every table with at least 3 rows of sample data.

## Part 2: Application Interface (UI)

You are required to build a functional interface with at
least 5 distinct forms.

### Navigation:

You must implement a seamless way to navigate between these forms.

### Functionality:

Each form must serve a specific purpose based on your project idea.

## Part 3: Database Interactions

Your application must successfully perform the
following operations:

- Select: Retrieve and display data in the UI.

- Insert: Add new records via a form.

- Update: Modify existing records.

- Delete: Remove records from the database.

## Submission Deliverables

### The SQL Script: A .sql file containing your CREATE

TABLE and INSERT INTO statements.

### The Application Source Code: Your full project folder.

### Project Documentation: A brief report detailing:

The chosen project theme.

A description of the 5 forms and
their specific functionalities.

A simple diagram or explanation
of how the forms navigate to one another.
